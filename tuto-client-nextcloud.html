Tutoriel : Installation et Paramétrage du Client Nextcloud

Guide détaillé pour installer et configurer le client Nextcloud sur votre ordinateur

Étape 1 : Téléchargement du Client Nextcloud

  1. Ouvrez votre navigateur web (Google Chrome, Mozilla Firefox, etc.).
  2. Accédez au site officiel de Nextcloud : https://nextcloud.com/install/#install-clients.
  3. Faites défiler la page jusqu'à la section intitulée "Desktop Clients".
  4. Cliquez sur le lien correspondant à votre système d'exploitation :
    • Windows : Cliquez sur "Windows Installer".
    • Mac : Cliquez sur "Mac Installer (Intel)" ou "Mac Installer (Apple Silicon)" selon votre modèle.
    • Linux : Suivez les instructions spécifiques pour votre distribution Linux.
  5. Le fichier d'installation sera téléchargé sur votre ordinateur. Notez l'emplacement du fichier (généralement dans le dossier "Téléchargements").

Étape 2 : Installation du Client Nextcloud

  1. Recherchez le fichier téléchargé dans votre dossier "Téléchargements".
  2. Faites un double-clic sur le fichier pour lancer l'installation.
  3. Suivez les instructions de l'assistant d'installation :
    • Acceptez les termes et conditions.
    • Choisissez l'emplacement d'installation (par défaut, c'est généralement correct).
    • Cliquez sur "Installer" ou "Suivant" selon les étapes.
  4. Une fois l'installation terminée, cochez l'option "Lancer Nextcloud" si elle est disponible, puis cliquez sur "Terminer".

Étape 3 : Configuration Initiale avec Authentification SSO

  1. Une fois le client Nextcloud lancé, une fenêtre de configuration s'affiche.
  2. Dans le champ "Serveur", entrez l'URL de votre serveur Nextcloud. Par exemple : https://cloud.perfo.fr.
  3. Cliquez sur "Suivant".
  4. Une fenêtre de connexion s'ouvre automatiquement, redirigeant vers App Perfo (https://app.perfo.fr). Vous devrez vous authentifier avec vos identifiants App Perfo, précédemment crées :
    • Entrez votre nom d'utilisateur ou votre adresse e-mail.
    • Entrez votre mot de passe.
    • Si vous utilisez l'authentification à deux facteurs (2FA), saisissez le code unique généré par votre application mobile.
  5. Après une authentification réussie, vous serez redirigé vers Nextcloud et connecté automatiquement.
Important : L'authentification se fait via App Perfo (https://app.perfo.fr) grâce au système de Single Sign-On (SSO). Assurez-vous d'utiliser les mêmes identifiants que ceux configurés dans App Perfo.

Étape 4 : Activation de la Synchronisation Virtuelle

  1. Une fois connecté, une nouvelle fenêtre apparaît pour configurer la synchronisation des fichiers.
  2. Pour éviter de consommer de l'espace disque local, activez la synchronisation virtuelle :
    • Cochez l'option "Synchronisation virtuelle" (ou similaire) dans les paramètres.
    • Cette fonctionnalité permet d'accéder à vos fichiers en ligne sans les télécharger physiquement sur votre ordinateur.
    • Les fichiers seront accessibles via l'explorateur de fichiers, mais ils ne seront téléchargés que lorsque vous les ouvrirez explicitement.
  3. Choisissez un emplacement local pour le dossier virtuel. Par défaut, un dossier "Nextcloud" sera créé dans votre répertoire personnel.
  4. Cliquez sur "Terminer" pour finaliser la configuration.
Conseil : Utilisez la synchronisation virtuelle pour économiser de l'espace disque local tout en ayant un accès rapide à vos fichiers. Évitez de synchroniser tous les fichiers localement, car cela peut ralentir votre ordinateur.

Étape 5 : Vérification et Utilisation

  1. Ouvrez l'explorateur de fichiers de votre ordinateur et naviguez jusqu'au dossier "Nextcloud".
  2. Vous verrez une liste de fichiers et dossiers disponibles sur le serveur Nextcloud.
  3. Double-cliquez sur un fichier pour l'ouvrir. Le fichier sera téléchargé temporairement et ouvert dans l'application associée.
  4. Les modifications apportées aux fichiers seront automatiquement synchronisées avec le serveur Nextcloud.

Conseils Importants

  • Gardez votre client Nextcloud ouvert : Pour garantir une synchronisation continue, assurez-vous que le client Nextcloud est toujours actif sur votre ordinateur.
  • Utilisez la synchronisation virtuelle : Privilégiez cette option pour éviter de consommer de l'espace disque local inutilement.
  • Mettez régulièrement à jour le client : Vérifiez périodiquement les mises à jour du client Nextcloud pour profiter des dernières fonctionnalités et corrections de sécurité.
